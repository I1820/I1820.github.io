--- layout: page title: History description: How it happend? ---

This project had been started as a python REST library for kaa but after project grew, we decided to complete it as stand alone platform.

In the begining we use SDN101 project as a good example of python REST API client. SDN101 is a great memory for Parham Alvani good old days.

When 18.20 decide to leave us alone, we created improved version of it, in this version we provide everything you need in simple package and the journey started here.

I1820 first was used in Aolab but after Aolab closed, it continues it life as stanalone project. After almost 1 year and many experiences new architecture is here. mainly new needs and considerations are based on ISRC platform project.
